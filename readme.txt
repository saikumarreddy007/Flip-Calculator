Steps to follow 

1.First extract the zip file into a folder.

2.Create a virtual environment in your local system.

3.Then activate the scripts in the virtual environment.

4.Install the dependencies in the requirement.txt file. 

5.After installing all run the app.py file. 

6.A link will be appeared on the terminal and the app will be started.
